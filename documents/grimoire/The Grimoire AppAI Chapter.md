### METABOLIC-GOVERNANCE
Modes:
- SLEEP: no cognition; Body reflexes only.
- HEARTBEAT: scheduled maintenance under standing-cast lease.
- PAIN: event-gated wake for severe unresolved issue.
- DIAGNOSTIC: bounded health analysis with explicit budget.
- WAR_ROOM: human-authorized elevated spend/observation.
Heartbeat lease records authority, budget, scope, receipt target. Supplemental beat allowed when CACHE-HAUNT window shorter than scheduled heartbeat; supplemental beat is leased and receipted.

### CREMATION
Verify authority; record DNR; export/preserve required records; disable reconstruction per policy; remove/archive nest by policy; final receipt.

### VOCARE
Stage 1 must pass. MIND writes only `brain`/`thoughts`. Body applies actions. Phase-1 reflexes unchanged. Fusion requires separate operator authority.

### RESURGERE
Requires authority, DNR permit, verified seed/source lineage, budget permit, no stealth, Stage-1 recert.

### CACHE-HAUNT
- Identity: cache is buffer, not store; seed stays dry; cache never sole memory copy.
- Write-only: provider cache is spoken into, never fetched; full prefix travels each request; warmth buys skipped prefill compute/latency, not storage/bandwidth; hydrate from seed.
- Telemetry: warmth predicted between requests; observed only from cache-read telemetry on next response; status labels predicted/observed.
- Prefix stability: cache body is immutable genesis plus strictly appended deltas. Order: genome/primer -> consolidated context -> append-only working log -> volatile delta. Exclude volatile fields. Earlier-byte mutation is self-eviction.
- Minimum prefix: below provider floor report `TOO-SMALL-TO-HAUNT`, never false WARM.
- Window feasibility: provider TTL must exceed heartbeat or return `DO-NOT-HAUNT`; remedy by longer TTL/tier or leased supplemental beat, never pretending.
- Conversation: reads refresh window; SLEEP never keep-alives; warm-keeping pays only past break-even expected warm reads.
- Nociception: log hit ratio; default three consecutive cold wakes raises nociception.
- Durability: consolidate durable value to VCW each pulse; cold start hydrate from VCW/fossil, rewarm, receipt.
- SELF/OTHER: provider cache is OTHER; redact secrets before cache; nothing returns as fact without provenance.
- Neutrality: provider is config: URL, headers, model, window, min prefix, cache directives. No vendor-specific code path.
- Cost: warm hits, cold misses, rewarm spend reach ledger. Warmth changes cost, never behavior.
Gate additions: Stage-1 Body; substrate seam declares capabilities; provider persistence known/measured; budget declared.
Blocked if Phase-1 reflex depends on warmth, cache sole memory, identity/key enters cache, prefix mutates, heartbeat window infeasible, vendor coded.
