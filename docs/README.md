# Mantle OS Docs Compatibility

Canonical Mantle OS documentation lives in [`../documents/`](../documents/).

This directory exists so older external links and LLM prompts that refer to `docs/` still
resolve to useful entry points.

- [`Mantle_for_Engineers.md`](Mantle_for_Engineers.md) points to the source-backed
  engineering on-ramp.
