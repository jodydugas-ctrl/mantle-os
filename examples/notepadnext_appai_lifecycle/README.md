# NotepadNext AppAI Lifecycle Example

This example is the compact MantleOS seed package for the NotepadNext
assimilation. The full working assimilation directory is intentionally not
checked in because it contains cloned host sources, runtime binaries, dependency
trees, and lifecycle run receipts that make the archive hundreds of megabytes.

- Contents:
  - `notepadnext_appai_lifecycle_spore.png`
  - `notepadnext_appai_lifecycle_egg.json`
  - `README.md`

The SPORE is the encrypted lifecycle seed. The egg JSON is the distilled app body
declaration: identity, host substrate, truths, commandments, app bands, controls,
surfaces, VCW commit policy, and primer. The README inside the archive explains
how the two files work together and why the full agent workspace is excluded from
Git.
